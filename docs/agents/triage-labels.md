# Triage Labels

工程技能使用五个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 实际使用的状态字符串。

| 技能中的角色 | 本仓库状态字符串 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求完整，适合 agent 独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，使用本表中对应的状态字符串。
