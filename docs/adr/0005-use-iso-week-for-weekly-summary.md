# ADR-0005: 周汇总使用 ISO 周

## 状态

已接受

## 背景

Expense Tracker v1 支持周汇总。周汇总必须明确一周的开始和结束，否则同一组交易在不同地区设置或不同实现下可能得到不同结果。

Java 提供 `DayOfWeek.MONDAY`、`TemporalAdjusters` 等日期 API，可以明确计算某一天所在周的周一和周日。

## 决策

v1 周汇总使用 ISO 周：

- 周一是一周开始。
- 周日是一周结束。
- `summary week --date YYYY-MM-DD` 汇总该日期所在的周一到周日。

## 后果

好处：

- 周汇总结果稳定，便于测试。
- 日期范围计算规则明确，不依赖系统区域设置。
- 命令参数只需要一个日期即可定位完整周范围。

代价：

- 不支持用户自定义一周从周日或其他日期开始。
- 如果后续支持地区化周起始日，需要引入新的配置和测试。
